# Holfuy Upgrader 1.0.0

## Initial Release

Holfuy Upgrader is an Android application for updating firmware on Holfuy weather stations over USB.

### Features

- Update Holfuy weather station firmware from an Android device using a USB OTG adapter.
- Guided firmware update workflow.
- Automatic detection of supported Holfuy weather stations.
- Firmware update progress indicator.
- Built-in User's Guide with setup instructions and troubleshooting tips.

### Supported Android Versions

Holfuy Upgrader has been tested on Android 8.1, 12, 15, and 16.

### Notes

- A USB OTG adapter is required.
- The weather station must be placed into ISP mode by powering it on immediately before tapping **Connect**.
- If the station does not connect, turn the station off and back on, then tap **Connect** again.
